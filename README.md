# Azure Data Factory CI/CD Automation - Project 2

## Description
Second iteration of **CI/CD pipelines for Azure Data Factory**, focusing on multiple environment deployments and pipeline versioning.

## Key Features
- Managed DEV, TEST, and PROD environments
- Automated pipeline deployments using GitHub Actions
- Integration runtime and dataset management
- ARM templates for infrastructure as code

## Tech Stack
- Azure Data Factory
- GitHub Actions
- JSON ARM Templates
- PowerShell / Azure CLI (if needed)

## Usage
1. Clone repository
2. Configure environment variables for Azure subscription
3. Run CI/CD workflows to deploy pipelines across environments

## Notes
This project reinforces skills in Azure DevOps and cloud data engineering best practices.
